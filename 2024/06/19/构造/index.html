构造题

发表于2024-06-19|算法笔记

Codefoces 1978C

CF 1978C 1500

  1. Statements

    定义长度为 $n$ 的排列 $p$ 的曼哈顿值 $V_p = \lvert p_1 - 1\rvert + \lvert p_2 - 2\rvert + \cdots + \lvert p_n - n\rvert$

    给定整数 $n$ 和 $k$，构造长度为 $n$ 的排列 $p$，使 $V_p = k$。

    $1 \le n \le 10^5$，$1 \le k \le 10^{12}$

  2. Solution

    Hint 1：如何计算 $V_p$ 的下限和上限？

    显然，$p = [1,2,\dots,n]$ 时，$V_p = 0$ 取到最小值；$p = [n,n - 1,\dots,1]$ 时，$V_p$ 取到最大值。

    Hint 2：$V_p$ 可以是奇数吗？

    假设 $p_i = x$，$p_j = y$，则它们对 $V_p$ 的贡献 $w = \lvert x - i \rvert + \lvert y - j \rvert$，

    $\operatorname{swap}(p_i,p_j)$ 后，$w = \lvert x - j \rvert + \lvert y - i \rvert$，$w$ 的奇偶性不变，因此 $V_p$ 只能是偶数

    容易想到一定存在排列 $p$，使 $k$ 可以取到 $0\sim k_{max}$ 范围内的任意偶数值。

    那么，应该如何构造排列 $p$ 呢？

    构造方法

    假设 $p = [1,2,\dots, n]$，若 $\operatorname{swap}(1,x)$，将会得到 $2\cdot \lvert 1 - x\rvert$ 的贡献。

    也就是说，$x\in [1,n]$ 时，可以得到 $0\sim 2\cdot (n - 1)$ 范围内的任意偶数值。

    若 $k > 2\cdot(n - 1)$，$\operatorname{swap}(1,n)$ 后，$1$ 和 $n$ 已经被固定，问题缩小至 $n - 2$ 的子问题。

    同理，若 $\operatorname{swap}(2,y),\ y\in [2,\ n - 1]$，可以得到 $0\sim 2\cdot (n - 3)$ 范围内的偶数值。

    根据数学归纳法，我们可以得到 $0\sim 2\cdot (n - 1) + 2\cdot (n - 3)$ 范围内的任意偶数…

    以此类推，最终有 $p = [n,\ n - 1,\dots,1]$，并且可以证明，任意 $k\in [0,k_{max}]$ 都可以被构造出来
